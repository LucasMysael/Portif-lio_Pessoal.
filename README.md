🚀 Funcionalidades
1. Accordion (Currículo)
Clique nos itens de formação, cursos e objetivo para expandir/recolher

Apenas um item fica aberto por vez

2. Slideshow (Projetos)
Cada projeto possui um carrossel de imagens

Transição automática a cada 3 segundos

3 projetos com descrições e tags de tecnologias

3. Animações
Barras de habilidades com animação ao entrar na tela (Intersection Observer)

Botões com efeito hover e transformação

Shapes flutuantes (float animation)

4. Responsividade
Layout adaptado para mobile (max-width: 768px e 900px)

Menu simplificado, colunas empilhadas

5. Smooth Scroll
Links internos com rolagem suave

🔧 Como Executar Localmente
Baixe todos os arquivos mantendo a estrutura de pastas

Coloque as imagens em assets/imagens/:

perfil.jpg (foto quadrada, recomendado 300x300px)

curriculo1.png e curriculo2.png (fotos do currículo escaneado)

Coloque o PDF em pdf_curriculo/Curriculo_Lucas.pdf

Abra o arquivo index.html no navegador

⚠️ Importante: Os caminhos das imagens e PDF estão configurados como ../assets/... e ../pdf_curriculo/.... Certifique-se de que os arquivos estejam nos lugares corretos.
